# miner
Filesize recursive data miner

Usage: miner -r [root directory] [-v [verbose level]]\n
 -r, --root [directory]      root directory
 -v, --verbose [level]       verbose level
                               0 - show nothing (default)
                               1 - show directories
                               2 - show directories and/or files
 -h, --help                  this page
